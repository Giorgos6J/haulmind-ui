# HaulMind Frontend

HaulMind Frontend is a React-based user interface for a fleet and trip management platform focused on fuel estimation, booking monitoring, trip analytics, and operational visibility.

The project is designed as a clean SaaS-style dashboard prototype for logistics and transport workflows.

## Features

- Dashboard overview with key operational metrics
- Vehicle and trip management pages
- Fuel prediction and trip result tracking
- Analytics and comparison views
- Clean UI structure for future backend integration

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Recharts
